# 🍀 QualVaiSer?

Gerador inteligente de números para as principais loterias da Caixa Econômica Federal, com base em frequência histórica real dos sorteios brasileiros.

🔗 **[Acessar o site](https://nobrudev.github.io/qualvaiser/)**

---

## 🎯 Loterias disponíveis

| Loteria | Números | Intervalo |
|---|---|---|
| Mega-Sena | 6 números | 1 a 60 |
| Lotofácil | 15 números | 1 a 25 |
| Quina | 5 números | 1 a 80 |
| Lotomania | 20 números | 0 a 99 |
| +Milionária | 6 números + 2 Trevos | 1 a 50 + 1 a 6 |

---

## ✨ Funcionalidades

- **Geração inteligente** — números gerados com base em frequência histórica de cada loteria
- **5 estratégias de seleção** — Inteligente, Quentes, Frios, Aleatório e Misto
- **Classificação visual das bolas** — quente, morno, frio e normal
- **Até 10 cartelas por geração**
- **Histórico de cartelas salvas**
- **Design responsivo** — funciona em desktop e mobile

## 🚀 Como rodar localmente

```bash

cd qualvaiser
```

Abra o arquivo `index.html` no navegador — sem necessidade de servidor ou dependências.

Este projeto é voltado para **entretenimento**. A frequência histórica é usada como referência, mas todos os sorteios de loteria são estatisticamente independentes. **Jogue com responsabilidade.**
